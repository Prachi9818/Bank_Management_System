# Bank Management System 🏦


A Bank Management System developed in Java with SQL database integration to simulate core banking operations. The project provides a simple, secure, and user-friendly interface for managing customer accounts and performing basic banking transactions.

This project was developed to strengthen my understanding of Java programming, Object-Oriented Programming (OOP), database management, and real-world application development.

---

📌 Features
--
- 🔐 User Registration (Sign Up)
- 👤 Secure Login Authentication
- 💰 Balance Enquiry
- ➕ Deposit Money
- ➖ Withdraw Money
- ⚡ Fast Cash Facility
- 📄 Mini Statement (Transaction History)
- 🗄️ SQL Database Integration
- ✅ Menu-driven Interface
